# Portfolio Project Guide


## Build & Dev Commands

- Install dependencies: `npm install`

- Development server: `npm run dev`

- Build project: `npm run build`

- Linting: `npm run lint`


## Tech Stack & Style

- **Framework:** Next.js (App Router)

- **Language:** TypeScript (Strict mode)

- **Styling:** Tailwind CSS (Mobile-first, Dark mode primary)

- **Components:** Modular, functional components with Arrow Functions.

- **Icons:** Lucide React or FontAwesome.


## Project Context

- This is a high-level Software Engineering portfolio for Jaret Eduardo.

- **Key Projects:** Inokaii (AutoML/C++/C#), KAIML (C++/CUDA/Rust Machine Learning library).

- **Current Role:** Backend Developer at API Inmobiliaria (.NET Core).

- **Design Goal:** Professional, minimal, performance-oriented (Engineering-first look).
